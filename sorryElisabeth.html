Sorry Elisabeth

Project type:

PC Game

Team:

Solo project

Game engine & Languages:

Godot, C++

Tools:

Git, Aesprite

You wake up as Elisabeth in a mysterious house that you don't recognise. You quickly understand that you and your mother have been kidnapped. You'll need to interact with the environment to understand why all of this happened, and find a way to get out of this house with your mom.

Interaction system

The player can interact with some objects in the room. Those specific objects detect if the player is close enough to be able to interact with them. The interactive objects contain a list of specific interactions like "Examine", "Read" or "Take" that are children of the abstract class "Interaction" (contains the abstract method "play").

Inventory system

The player has an inventory that contains cells, and a cell can contain an item which inherits from an interactive object class. The player can interact with items directly in the inventory as well. Items can have specific interactions like "combine" or "use".
