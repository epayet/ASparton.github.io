Retro 1010!

Project type :

Personal, solo project

Graphic library & Languages :

Pygame, Python

Tools :

Git, Aesprite

Retro 1010! is a 2D arcarde game inspired by 1010!. You need to choose between diferent pieces to place on a ten by ten board. You earn points each time that you place a piece, or suppress a row or a column by filling all its ten cells. You loose when there is no more places for any available piece on the board. Done in my high school's free time, I spent a good time developing in solo, all parts of the game, including the sound system and the texture design. It would be a fun challenge to try to improve the code 1 year after !

Piece's place verification
